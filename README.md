# Obstacle Detection Car

## Description
A robotics project that detects obstacles using an ultrasonic sensor and detects collisions.

## Technologies Used
- Arduino
- Ultrasonic Sensor
- Servo Motor
- Embedded C

## Features
- Detects nearby obstacles
- Suggests change of direction with farther obstacle to avoid collisions
- Real-time distance sensing
